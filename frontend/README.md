# Frontend (Phase 2)

Next.js chat UI for the personal finance agent. **Not yet implemented** — this
directory is a placeholder for Phase 2.

Planned:

- Chat panel that calls `POST /api/chat` and renders streaming replies.
- Upload dropzone wired to `POST /api/upload`.
- Transaction table with filters by date / category.
- Tool-call trace expandable below each agent reply (great for demos).

Stack: Next.js (App Router) · TypeScript · Tailwind · shadcn/ui.
